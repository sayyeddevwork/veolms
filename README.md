# VeoLMS

A production-grade Learning Management System (LMS) inspired by modern e-learning platforms such as Udemy. This project is being built with a strong focus on software architecture, security, scalability, observability, maintainability, and engineering best practices.

---

## 🚀 Project Goals

- Production-grade system design
- Clean and maintainable architecture
- Secure authentication and authorization
- Real payment integration (Razorpay Test Mode)
- Video-based learning platform
- Comprehensive testing strategy
- Cost-effective cloud deployment
- Professional documentation

---

## 🛠️ Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- TanStack Query
- Zustand

### Backend

- Node.js
- Express.js
- TypeScript
- Prisma ORM
- PostgreSQL

### Infrastructure

- Neon PostgreSQL
- Cloudinary
- Razorpay (Test Mode)
- Render
- Vercel
